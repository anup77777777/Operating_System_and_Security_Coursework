# About This Repository

This repository contains my coursework for the **ST5004CEM – Operating Systems and Security** module. The project demonstrates the implementation of fundamental operating system concepts using the **C programming language** on Ubuntu Linux.

Each task focuses on a different area of operating systems, including process management, memory management, file systems, security, networking, and inter-process communication.

# Assignment Tasks

## Task 1 – Process Management and Threading

**Current Status:** ✅ Completed

### Features

- Process creation using `fork()`
- Parent and Child process execution
- Multi-threading using POSIX Threads
- Three concurrent worker threads
- Round Robin Scheduling Simulation
- Thread Synchronization using Mutexes
- Condition Variables for scheduling
- Race Condition Prevention
- Deadlock Prevention through ordered resource locking
- Interactive terminal-based monitoring dashboard
