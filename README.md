# Repository Overview

This repository contains my coursework for the **ST5004CEM – Operating Systems and Security** module.

The coursework demonstrates the practical implementation of fundamental operating system concepts using the **C programming language** on **Ubuntu Linux**. Each task focuses on a different area of operating systems, including process management, memory management, file systems, security, network programming, and inter-process communication.
